About

What is ImgReader ?

ImgReader is a free images reader web application that makes your images content to become easier to read.

Features :

  1. Secured from an automatic content stealer like crawler, scrapper, web proxy, etc.
  2. You control the image source
  3. Lightweight and Fast Loading
  4. Private but Shareable
  5. Free Forever

How to use

1. Generate Json Source

This is to generate the json data source to be uploaded to your webserver.

Fields are marked with * is required! Images must be separated by new line.
Additional Information HTML is not allowed in description.

2. Generate Link

After you have the link of your json data source, you must generate it to hash the link.

Fields are marked with * is required!

Docs

Requirement Web Server

The requirement of web server to upload the json source is :

  • Request HTTP method is GET.
  • CORS must be allowed for .
  • Response Content-Type must be application/json.

If you don't want to use web server, as alternative, you are able to upload your json source into GitHub, GitLab, CDN, etc.
